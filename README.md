# Instructions  

Please run all Functions inside your main method once.  They should be in the order of the questions provided.

1. Create a method call intChecker()</br>
It takes no parameters.</br>
Ask the user to "Input an integer: ".</br>
If the input is an integer, return the integer.</br>
Otherwise ask the user again.</br>
</br>

2. Create a method call doubleChecker()</br>
It takes no parameters.</br>
Ask the user to "Input a number: "</br>
If the input is any kind of number, return it.</br>
Otherwise ask the user again.</br>
</br>

3. Create a method call booleanChecker()</br>
It takes no parameters.</br>
Ask the user to "Input a boolean: "</br>
If the input is a boolean, return the boolean.</br>
Otherwise ask the user again.</br>
</br>

4. Create a method call multipleIntChecker()</br>
It takes no parameters.</br>
Ask the user to "Input an integer: " 4 times.</br>
If the input is not an integer, ask again.</br>
return true after they have input 4 integers.</br>
